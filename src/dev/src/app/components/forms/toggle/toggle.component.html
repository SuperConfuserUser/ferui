Input toggle component overview

Be sure to check here here for a general overview of FerUI form component usage.

Basic toggle

By simply applying the fuiToggle directive to a checkbox input element, the input will be styled as a FerUI Toggle and behave like a checkbox. There is no label to describe the field.

Labeled toggle

Wrapping the input with the designated fuiToggleWrapper allows creation of a labeled toggle. The input and label elements may be placed in any order within the wrapper.

Toggle with validation

To validate that a toggle is toggled (checked) and add style accordingly, apply the required validator directive and wrap everything in fuiToggleContainer. In the absence of the fuiControlError element, the default error message will show. A custom error message may be specified via the innerHTML of fuiControlError.

Disabled toggle

A disabled toggle input will have disabled functionality as expected as well as styling to reflect its state.
